
# JoyLearning AI培训管理后台 - SaaS化实施方案

## 一、产品定位与SaaS化架构

### 核心定位
打造**多租户AI培训SaaS平台**，支持企业级客户独立管理培训体系，实现"教、学、练、考、评"五维一体的智能培训管理。

### 多租户架构设计
```
超级管理员（平台级）
└── 企业租户（Organization）
    ├── 部门（Department）
    │   └── 团队（Team）
    └── 统一配置：套餐、配额、品牌定制
```

---

## 二、核心功能模块

### 1️⃣ 租户与权限管理
- **企业入驻**：企业自助注册、管理员设置
- **组织架构**：支持导入/手动维护部门树
- **角色权限**：超级管理员、业务管理人、普通学员三级权限
- **数据隔离**：各租户数据完全隔离，部门级数据范围控制

### 2️⃣ 培训计划管理
- **计划创建**：设置培训名称、目标、时间周期、关联部门
- **章节编排**：拖拽式章节管理，支持教学+练习+考评组合
- **内容关联**：章节可关联视频、文档、PPT等教学资源
- **状态管理**：草稿 → 待发布 → 进行中 → 已结束

### 3️⃣ 练习管理
- **练习场景配置**
  - 自由对话模式（本期重点）
  - 固定剧本模式（预留）
- **剧本设计**：
  - AI辅助生成练习剧本（基于场景描述）
  - 配置学员身份、AI角色身份
  - 设置练习目标与评分标准
- **对话设置**：次数限制、时长限制、评分权重

### 4️⃣ 考评管理
- **对话式考评**：对接AI面试能力，配置考核SOP
- **问卷式考评**：题库管理、问卷组卷（预留接口）
- **评分标准**：多维度打分权重设置

### 5️⃣ 学员管理与邀请
- **邀请方式**：
  - 批量Excel导入
  - 按部门一键邀请
  - 单人表单邀请
- **学员状态追踪**：未开始 → 进行中 → 已完成
- **学员详情**：查看学习进度、练习记录、考评结果

### 6️⃣ 学员成长地图 ⭐ 新增亮点
- **岗位能力模型**：
  - 可视化技能树（专业能力、胜任力）
  - 各技能点权重与达标标准
- **能力成长雷达图**：多维度能力可视化
- **学习路径推荐**：基于能力短板智能推荐培训内容
- **成长历程时间轴**：记录关键节点与能力提升

### 7️⃣ 数据看板与报告
- **培训看板**：
  - 计划完成率、学员参与率、平均学习时长
  - 趋势曲线图、培训漏斗图
- **练习看板**：练习完成情况、平均得分分布
- **学员报告**：个人成绩单、详细练习/考评记录
- **导出功能**：支持Excel多Sheet导出

### 8️⃣ AI角色管理
- **数字人形象配置**：头像、名称设置
- **语音设置**：预留语音风格选择
- **能力关联**：角色与知识库、话术的关联配置

### 9️⃣ 内容与知识库管理
- **知识库**：支持上传文档、话术管理
- **内容解析**：上传PDF/PPT自动提取内容（预留）
- **题库管理**：考评题目的增删改查

## 三、SaaS化增强特性

### 订阅与计费
- **套餐体系**：基础版、专业版、企业版

- **用量统计**：实时消耗监控与预警

### 品牌定制
- **Logo自定义**：企业可上传自有品牌Logo
- **主题色配置**：支持主色调自定义
- **域名白标**：（高级功能预留）

### 数据安全
- **租户隔离**：RLS策略确保数据安全
- **操作日志**：关键操作可审计追溯
- **数据导出**：支持租户数据一键导出

## 五、技术实现方案

### 前端框架
- React + TypeScript + Tailwind CSS
- shadcn/ui 组件库（专业B端风格）
- Recharts 数据可视化

### 后端能力
- Supabase（数据库 + 认证 + RLS）
- Edge Functions（业务逻辑 + AI接口预留）

### 数据模型核心表
- organizations（租户/企业）
- departments（部门树）
- training_plans（培训计划）
- practice_sessions（练习场景）
- assessments（考评配置）
- trainees（学员）
- skill_models（能力模型）
- training_progress（学习进度）
